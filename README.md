# Mortgage Ready Coach Starter

This is a deployable Next.js starter project for Mortgage Ready Coach.
Updated deployment trigger.

## Local setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Create a GitHub account if you do not already have one.
2. Create a new GitHub repository named `mortgage-ready-coach`.
3. Upload these project files.
4. In Vercel, choose **Import Git Repository**.
5. Select the repository.
6. Click **Deploy**.

## Domain

After deployment, connect MortgageReadyCoach.com in Vercel Project Settings > Domains.
